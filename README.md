# Customer Churn Prediction

## 📌 Project Overview
This project analyzes customer data to identify factors contributing to churn and builds a predictive model to classify customers likely to leave.

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Build a predictive model
- Generate actionable insights

## 🛠️ Tools Used
- Python (Pandas, NumPy)
- Matplotlib / Seaborn
- Scikit-learn

## 📊 Key Steps
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Building
5. Evaluation
